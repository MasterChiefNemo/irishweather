The aim of this project has been to use Javascript alongside HTML and CSS, to gather weather data. Users can retrieve data for a specific location, by simply using the search functions available.

Client Side Assignment 2

About This Project

On the index page, the user can search for weather forecast data, for any location of their choosing within Ireland. The information returned is the current temperature, maximum and minimum temperatures, humidity, wind speed and direction. Accompanying the data, is the users current location.

On the comparison page, a user can compare the temperatures for two locations over a period of time. The information provided, is returned and graphed visually for a users ease of use. All data, at the users choice can be returned in metric or imperial units, on both pages.
