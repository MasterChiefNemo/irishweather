Open Weather Map Data

Assingment 2 - Gathering Data

Welcome to Assignment 2 for Client Side Technologies. In this project, I will be making use of HTML, CSS and Javascript to obtain data from Open Weather Map.

In the main body, to the right of this text, you will see a text field, radio buttons and a submit button near the top of the page. This allows you to search and retrieve weather data for any town or city in Ireland. The output is shown to the right of the input box. As for the map, it shows the users current location.

Why only Ireland? For the purpose of this assignment and time constraints, it is easier to allow a user to search only Irish locations and compare weather information. It can be extended to allow for other Countries quickly.
